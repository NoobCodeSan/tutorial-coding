git status - текущее состояниие

git add - выделение файла для отслеживания в репозиторие

git commit -m "Текст" - команда сохранение изменений (в истории)

git push origin HEAD - отправить в удаленный репозиторий (HEAD - это значит все до последнего)

git log --oneline - показать историю commit-ов

git pull origin - синхронизироваться с удаленным репозиторием
